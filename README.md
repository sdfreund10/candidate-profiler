## Candidate Profiler

Crawls a political candidate's website to build a quick politcal profiler. Streamlines candidate research.

## Tech

- Sinatra web server and a basic HTML/CSS frontend enhanced with Turbo.
- Web pages are pulled, converted to markdown, and analyzed via LLM.
- Site scraping is entirely directed by an LLM agent.

### Ideas

- Pull the whole sample,ballot from somewhere on the internet
- Have the user paste in the the campaign websites for each race
- Generate a packet
